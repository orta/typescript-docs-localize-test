TypeScript는 여러 언어를 지원합니다. 하지만 `tsconfig.json`를 통해 설정 할 수 없습니다. 오직 host의 envrionment 혹은 CLI flag인 `--locale`로 설정 할 수 있습니다. E.g. `tsc --locale ru`.

### TypeScript 3.7 에서 지원되는 Locale

| Name              | Locale  |
| ----------------- | ------- |
| 브라질 포르투갈어 | `pr-BR` |
| 중국어 (간체)     | `zh-TW` |
| 중국어 (번체)     | `zh-CN` |
| 체코어            | `cs-CZ` |
| 영어 (US)         | `en-US` |
| 프랑스어          | `fr-FR` |
| 독일어            | `de-DE` |
| 이탈리아어        | `it-IT` |
| 일본어            | `ja-JP` |
| 한국어            | `ko-KR` |
| 폴란드어          | `pl-PL` |
| 러시아어          | `ru-RU` |
| 스페인어          | `es-ES` |
| 터키어            | `tr-TR` |
